## ✨ Features:

### 1.🔐 Authentication System
  1.Role-based login/signup (Student / Faculty)
  2.Secure credential handling (UI level)

### 2.📊 Student Dashboard
  1.Access study materials
  2.View important notices

### 3.👨‍🏫 Faculty Panel
  1.Upload materials
  2.Manage notices

### 4.🎨 Modern UI/UX
  1.Gradient design
  2.Clean layout
  3.Smooth navigation
